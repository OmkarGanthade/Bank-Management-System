# Bank Management System

A console-based Bank Management System developed in C++ to strengthen object-oriented programming concepts and modern C++ practices. The project follows a modular architecture with separate classes for account management, transaction handling, and file operations.

## Features

- Create Bank Account
- Deposit Money
- Withdraw Money
- Delete Account
- View All Accounts
- Transaction History
- File-based Data Persistence
- Input Validation
- Automatic Account Number Generation

## Concepts Demonstrated

- Object-Oriented Programming (OOP)
- Encapsulation
- Composition
- STL (`std::vector`)
- File Handling (`fstream`)
- Multi-file Project Structure
- Constructor Initializer Lists
- Static Data Members
- Input Validation

## Technologies

- C++17
- STL
- Visual Studio Code
- g++
